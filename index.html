Kees's Futoshiki Helper v1.0

Solvable

Rules

  • Fill each cell with a digit from 1 to N (where N is the grid size)
  • Each digit must appear exactly once in every row
  • Each digit must appear exactly once in every column
  • Inequality constraints (triangles) must be satisfied: the wide side points to the larger number

Controls

  • Enter digits 1 to N in each cell
  • To add a constraint, select the cell with the larger digit and press L, R, T, B (Left, Right, Top, Bottom) to point toward the smaller digit
  • Press Delete or Backspace to clear a cell
  • Use Arrow keys to navigate between cells
  • Enable Auto Digits to see possible values for each cell
  • Click Hint to highlight a cell that can be solved
